# virgil

A universal language for animations and visualization.

Compiles into C++ and Javascript, with special integrations for use
with the [Vizify](https://git.corp.yahoo.com/lpstein/vizify) platform.

## Usage

**Installation**

Do it manually!

**Compiling**

```bash
vzs-js FILE.vgl
```

**Testing**

```bash
npm test
# Or
./script/test.sh
```
